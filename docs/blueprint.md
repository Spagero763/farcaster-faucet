# **App Name**: Farcaster Faucet

## Core Features:

- Wallet Connection: Connect to Farcaster account using Wagmi and RainbowKit.
- Balance Display: Display balance information with units of ETH
- Username Input: Allow users to input a Farcaster account username.
- Tip Amount Entry: Allow users to enter a tip amount
- Transaction Dispatch: Provide a button for the user to dispatch a transaction on chain.
- Transaction Confirmation: Notify the user as the transaction is being confirmed

## Style Guidelines:

- Primary color: Vibrant purple (#A06CD5) to reflect innovation in web3.
- Background color: Light gray (#F0F2F5) for a clean, modern look.
- Accent color: Electric blue (#7DF9FF) to highlight key interactive elements.
- Font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text
- Use simple, geometric icons from a set like Phosphor to maintain a clean aesthetic.
- Employ a modular layout with rounded cards to provide a structured, touch-friendly interface.
